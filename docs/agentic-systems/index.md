## Authoritative references

Use the following standards and guidance to validate implementation choices as the ecosystem evolves:

- [Model Context Protocol specification](https://modelcontextprotocol.io/specification/2025-06-18) — the protocol requirements for connecting AI applications to tools, prompts, and resources.
- [A2A Protocol specification](https://a2a-protocol.org/latest/specification/) — interoperability, discovery, task, and security requirements for agent-to-agent communication.
- [OpenTelemetry GenAI semantic conventions](https://opentelemetry.io/docs/specs/semconv/gen-ai/) — the evolving common vocabulary for tracing and measuring generative AI systems.
- [OWASP prompt-injection guidance](https://genai.owasp.org/llmrisk/llm01-prompt-injection/) — a practical security baseline for untrusted instructions and agent workflows.
