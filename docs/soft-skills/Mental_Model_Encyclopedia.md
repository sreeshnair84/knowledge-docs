# The Mental Model Encyclopedia

A Full-Depth Reference for Executive Cognition, Systems Thinking, Strategic Analysis & Counterfactual Thinking

68 frameworks · full-depth reference, split across 4 parts

Each framework entry in this encyclopedia follows the same field structure — Purpose/Core Idea, Origin & Creator, Typical Users, Inputs→Outputs, Strengths, Weaknesses, Best Use Cases, Failure Modes, Enterprise Example, AI Angle, Prompt Template, and Related Frameworks — so any entry can be scanned or compared consistently regardless of which part it lives in. This page is the series index; the full content lives in the 4 parts below.

## Series structure

| --- | --- | --- | --- |
| 1 | [Executive Cognition Frameworks](Mental_Model_Encyclopedia_Part1_Executive_Cognition) | The raw cognitive habits and dispositions — cognitive bravery, intellectual humility, second/third-order thinking, calibration, probabilistic and Bayesian reasoning — that determine how well any downstream framework gets used. | 19 |
| 2 | [Systems Thinking Frameworks](Mental_Model_Encyclopedia_Part2_Systems_Thinking) | Tools for reasoning about problems with no single cause or linear solution — feedback loops, leverage points, the Cynefin framework, wicked problems, and stock-and-flow modeling. | 17 |

| 4 | [Counterfactual Thinking Frameworks](Mental_Model_Encyclopedia_Part4_Counterfactual_Thinking) | Tools for deliberately reasoning against the current plan — inversion, devil's advocate, kill criteria, pre-mortem-adjacent failure mapping, and anti-goals — to surface risks that confirmation-seeking planning misses. | 10 |

## Who should read which part

**Executives and decision-makers** sharpening personal judgment start with **Part 1** — the cognitive dispositions apply regardless of which analytical framework gets used downstream.

**Enterprise architects and organizational designers** diagnosing recurring, structural problems want **Part 2** — feedback loops, leverage points, and the Iceberg Model explain why symptom-level fixes keep failing.

**Corporate strategists** running a planning cycle — market entry, competitive response, portfolio prioritization — want **Part 3**, the largest part, covering the standard toolkit from SWOT through decision trees.

**Anyone finalizing a high-stakes, hard-to-reverse plan** should close with **Part 4** before committing — these frameworks exist specifically to surface what a confirmation-seeking process misses.
